### Colors
This library has predefined colours. Selector `:root` defines those colours.You can change them to whatever you like.

### Headings 
You can use headings from `<h1>` to `<h6>`. Font sizes are: 
* 40px for `<h1>`
* 32px for `<h2>`
* 24px for `<h3>`
* 20px for `<h4>`
* 18px for `<h5>`
